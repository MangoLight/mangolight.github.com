Revealery

Introduction

Revealery is a simple jQuery plugin that makes slowly appears in a random order the children of an element.



Installation

Revealery requires jQuery to be running on your site :

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js"></script>

So, to install Revealery, simply download it and add these lines onto your code :

<script type="text/javascript" src="jquery.revealery.min.js"></script>


Options

Parameter name Explanation Default value
beginning_delay Delay before starting to make elements appear (in ms) 300
delay_between Delay between two apparitions (in ms) 150
duration Duration of each apparition (in ms) 500
recursive Apply Revealery on the children of the object false


Examples

Basic usage :

$('#album').revealery();
Run

With options :

$('#album').revealery({
   'beginning_delay' : 0,
   'delay_between' : 20,
   'duration' : 250
});
Run

With a callback :

$('#album').revealery({
   'beginning_delay' : 700,
   'delay_between' : 0,
   'duration' : 1000
},function(){
   alert('Finished');
});
Run

Recursive usage

The recursive option allows to call Revealery for each children. The result can be a bit suprising !

$('body').revealery({
   'recursive' : true
});
